# Excel一括盤面出力 追加版

## 追加機能

左サイドバーに「Excel一括盤面出力」パネルを追加しました。

できること:

- 複数のExcelファイルを同時に読み込み
- 全シート横断で盤面出力対象を抽出
- 以下の条件でAND絞り込み
  - pick_M1
  - S
  - H
  - 使用キャラE
  - 使用者E
  - キーワード
- 条件欄はカンマ区切りで複数指定できます
- 一覧行をクリックすると現在の戦術ボードに反映
- PNGはzipで一括出力
- PDFは1試合1ページで一括出力
- 「暗号機配置」列がない、空欄、不正値の場合は第1組を使用

## 使い方

1. zipを展開します。
2. `idv-board` フォルダで以下を実行します。

```bat
npm install
npm run dev
```

3. ブラウザで表示されたローカルURLを開きます。
4. 左サイドバーの「Excel一括盤面出力」からExcelファイルを読み込みます。
5. 条件を指定し、該当行を確認します。
6. 必要に応じて行クリックで盤面プレビューします。
7. `PNG zip出力` または `PDF出力` を押します。

## 注意

- 一括出力はブラウザ処理のため、上限は先頭200件にしています。
- 出力対象が多い場合は条件で絞り込んでください。
- 追加依存関係として `jspdf` と `fflate` を使用しています。
- Viteの実行権限問題が出る環境では、通常はWindowsのcmdで `npm run dev` を実行してください。


## 2026-05-05 追加仕様

- `使用キャラE` をカンマ区切りで複数指定した場合、検索結果・PNG出力・PDF出力は入力順で並びます。
  - 例: `オペラ,隠者,アイヴィ` → オペラ → 隠者 → アイヴィ
- PNG zip内のファイル名はフォルダ分けせず、先頭に連番、その次に `使用キャラE` を入れます。
  - 例: `001_オペラ_2025-...png`
- 同じ `使用キャラE` 内では、日付 → R → 前後半 → Home/Away → 元データ順で並びます。
